# Design References

These files are external reference systems used to guide ThreadTrace interface design. They are not the product design system.

## Current Direction

ThreadTrace should use a Notion + Apple reference blend:

- Notion for approachable workspace structure, clear blocks, light surfaces, and low-friction everyday use.
- Apple for premium spacing, product-first presentation, typography discipline, and restrained interaction color.

Use the root `DESIGN.md` as the source of truth for ThreadTrace. Read these files only when visual direction, page polish, or frontend redesign work needs extra context.
